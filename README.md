QULIN – Nutritional Analysis Module
📌 Overview

QULIN is an AI-driven smart kitchen assistant designed to reduce food waste and improve healthy eating. The Nutritional Analysis module helps users understand the nutritional value of their meals and make better dietary decisions.

🚀 Features
🍎 Food Nutrient Breakdown – Displays calories, proteins, fats, carbs, vitamins, and minerals
🧠 AI-Based Analysis – Estimates nutrition from ingredients or recipes
⚠️ Allergen Detection – Identifies potential allergens in food
🎯 Personalized Insights – Suggests healthier alternatives based on user preferences
📊 Diet Tracking – Helps monitor daily nutritional intake
🛠️ Tech Stack
Frontend: React.js / Flutter (UI)
Backend: Node.js, Express.js
Database: MongoDB
AI/ML: Python (for prediction & analysis)
APIs: Nutrition APIs (e.g., Edamam / Spoonacular)

# 🥗 Nutrition AI Chatbot

An AI-powered Nutrition Chatbot that provides accurate, dataset-backed answers to food and diet-related questions using local embeddings and FastAPI.

---

## 🚀 Features

- Nutrition Q&A (calories, macros, foods)
- FAISS vector search
- Local embeddings (no OpenAI cost)
- FastAPI backend
- Dataset-driven answers
- Scalable architecture

---

## 🏗️ Tech Stack

- **Python**
- **FastAPI**
- **FAISS**
- **Sentence Transformers**
- **Perplexity API (optional)**
- **Pandas**
- **Uvicorn**
